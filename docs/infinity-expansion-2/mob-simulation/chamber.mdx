# Mob Simulation Chamber

Mob Simulation Chamber is an electric machine that takes [Mob Data Cards](/infinity-expansion-2/mob-simulation/card) and produces the drops of the mob and provides experience over time.

## Usage

Put a Mob Data Card in the input slot (top left) and connect it to an energy network.

<ImageGallery images={[
  { src: '/images/infinity-expansion-2/mob-simulation-chamber.png', alt: 'Mob Simulation Chamber GUI', sizePercent: 50 }
]}/>

Click on the **Experience** icon (top right) to collect the experience.

## Information

**Energy consumption**: 150 J/t

The actual energy consumption will also add the energy consumption of the Mob Data Card.

:::info default values

Some of these information can be changed in the [item specific configurations](/infinity-expansion-2/config/items) by server owners. The wiki only shows the default values. Check the actual information in-game.

:::
